# core


## How to build

```bash
$ mkdir -p out
$ cd out && cmake ../
$ make

or

$ ./build.sh
```


## How to execute

  ```bash
  $ cd out
  $ sudo ./sdpc
  ```
* If not executed, please check the [uart path](https://gitlab.com/MAZE-dankook/self-driving-patrol-car/-/blob/main/core/common/sdpc_uart.c)
